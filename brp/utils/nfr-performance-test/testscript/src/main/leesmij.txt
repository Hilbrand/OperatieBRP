1) Genereer en laadt de database (eenmalig!)

    importeer nfr-performance-test/belastingprofiel1/src/main/resources/backup/50kpersonen in de database

    @pdb97.modernodam.nl/nfr_profiel1

2) Install de NFR-load-generator.jar

  > mvn clean install -f nfr-performance-test/load-generator/pom.xml


3) Verkrijg de benodigde BRP deployables:

   > mvn clean install

   De artefacten bevinden zich in target/deployables


4) Deploy de artefacten op de omgeving*

   > ./deployscript.sh


5) Start de performancetest

   > ./startload.sh



*   Omgeving instellingen zijn gedaan in het omgeving.sh script
    Van belang is dat SSH niet telkens om je wachtwoord vraagt, zie http://www.linuxproblem.org/art_9.html

**  Applicatie logs kunnen snel bekeken worden met het logs.sh script
