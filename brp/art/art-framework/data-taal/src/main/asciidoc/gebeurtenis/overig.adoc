==== Beëindig adres
In sommige situaties wil je het adres van een persoon beëindigen. Denk bijvoorbeeld aan de situatie waarin een persoon is vertrokken en het onbekend is
waarheen.

[source,groovy]
----
beeindigAdres() {
}
----

De datum waarop het adres beëindigd dient te worden, kan meegegeven worden.
[source,groovy]
----
beeindigAdres() {
   op 20010911 <1>
}
----
<1> De einde datum kan als getal worden meegegeven.
