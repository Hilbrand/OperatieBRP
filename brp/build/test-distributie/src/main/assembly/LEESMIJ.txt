1)  Pak de brp-test-distributie-VERSIE-zip uit.

2)  Voer het testclient/database/93-deleteCascade.sql script uit op de database.

    Dit voegt cascade delete constraints toe aan de tabellen zodat
    personen in de tests gemakkelijk verwijderd kunnen worden.

3)  Voer het testclient/database/art-testdata.sql script uit op de database.

    Dit voegt de testvulling toe.

4) Pas testclient/landscape.groovy aan om eventueel de tomcat / db / activemq host aan te passen

5) Voer het start.sh script uit.

9) De tests worden nu uitgevoerd. Testresultaten komen in de jbehave en funqmachine folders te staan.
